### 环听雪：玉环清脆之声与雪落梅枝的簌簌清响相和。
行香子·环听雪

潮琢环声，月泻椰庭。
认桄榔、暗结盐晶。
玳瑁珠跳，珊瑚光烹。
恰三分露，七分霰，十分清。

蕉衫漉酒，蚺盅盛星。
笑坡仙、嚼蜡调羹。
蓼衣烟碎，鹊尾云平。
剩一痕霜，一瓯月，一宵冰。
